# Product Direction

FlowState is operational runtime infrastructure for AI-assisted environments.

Treat reusable context as durable runtime material, not disposable prompt text.

Optimize for:

- operational clarity
- reproducibility
- provenance
- local-first ownership
- human inspectability
- provider-neutral retrieval

Avoid:

- chat-first UX
- prompt marketplace framing
- agent orchestration
- autonomous execution
- hype-driven AI productivity language
