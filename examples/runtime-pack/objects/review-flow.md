# Code Review Flow

Review for:

- correctness
- data loss risks
- security and trust boundaries
- portability regressions
- missing tests
- confusing operational behavior

Lead with concrete findings.

Prefer file and line references when available.

Do not bury serious issues under summaries or compliments.
